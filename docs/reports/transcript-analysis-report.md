# Transcript Analysis Report: SOP Extraction Guide

**Date**: 2026-01-18
**Total Transcripts**: 297
**Purpose**: Comprehensive guide for digesting and extracting SOPs from each Lenny's Podcast transcript

---

## Executive Summary

This report provides a systematic approach to extracting Standard Operating Procedures (SOPs) from 297 Lenny's Podcast transcripts. Each transcript is categorized by domain expertise, with specific guidance on what SOPs to extract and how to process the content.

---

## Processing Methodology Per Transcript

### Standard Processing Steps

For each transcript:

1. **Initial Scan** (5 min)
   - Identify the guest's primary expertise
   - Note recurring themes and frameworks
   - Mark timestamps of actionable content

2. **Deep Analysis** (15-30 min)
   - Extract explicit frameworks/methodologies
   - Identify step-by-step processes
   - Capture decision-making heuristics
   - Note anti-patterns and warnings

3. **SOP Drafting** (10-20 min per SOP)
   - Structure per template in guide.md
   - Add context and triggers
   - Include prerequisites and pitfalls

4. **Validation** (5 min per SOP)
   - Verify actionability
   - Check self-containment
   - Confirm proper attribution
